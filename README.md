# Web_OrderSheet
>  웹, 앱 으로 보여지는 주문 차트( 카운터 주방 요청사항) 
> web 은 server 주문
> app 은 client 확인 완료

## 기능
> Web 주문 관리 (주문 추가, 삭제, 순서 변경)

> app client는 (주문 확인, 완료)

## 사용언어
> React , Kotlin

## 개발환경 셋팅 및 React 설치 
> 1. node v16.15.1 설치  
> 2. vs code 설치  
> 3. 프로젝트 생성
>> ``` npx create-react-app 프로젝트명 ```  
>> ``` cd 프로젝트명 ```  
>> ``` npm start ```
